List_Shift
==========

**Extension for Python List Class**

Installation
------------

``` html
pip install list_shift
```

Get Started
-----------

``` html
from list_shift import List

arr = List([1,2,3])
arr.shift()
#=> 1
arr.shift()
#=> 2
arr.shift()
#=> 3
arr.shift()
#=> None
```
